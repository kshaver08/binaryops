Binary Logic Operators

There are certain operators one can use when dealing with logic in mathematics.

Operators

  • AND (&&): returns true only if both operands are true, otherwise returns false.
  • OR (||): returns true if either operand is true; if both are false, returns false.
  • NOT: returns the opposite of the operand.
  • XOR: returns true only if one operand is true; returns false if none or both are true.
